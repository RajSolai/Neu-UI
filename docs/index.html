<link rel="stylesheet"
href="https://rawcdn.githack.com/RajSolai/Neu-UI/860d56b1a0dd3a559b5ae56acb0bda74cfa39439/dist/neu-ui.css>

Classes CheatSheet

Get to know the Classes of the Neu-UI stylesheet 👩🏻‍💻

Class Name

Functionality
btn Form component button
container Creates a responsive fluid container
cont-fluid Create a Flexible container
card Parent tag for the card
card-img Creates image for the card component
card-header Creates table Heading (Text)
card-content Creates content for the card
